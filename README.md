## 英語学習チャットアプリ　Languchat
概要
チャットアプリとフレーズ集を組み合わせたアプリ。会話の中で学んだフレーズなどを登録して、メッセージ作成時に参照できる。
メーっセージ作成時日本語⇨英語の翻訳
